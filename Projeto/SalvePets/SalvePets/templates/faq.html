{% trans "Quem pode realizar uma adoção?" %}

{% trans "Qualquer pessoa, plenamente capaz, maior de 18 anos, mediante assinatura do contrato de adoção." %}

{% trans "Os animais que vão para adoção são castrados?" %}

{% trans "Sim. Os animais que participam do processo de adoção são castrados, vacinados de acordo com a idade, vermifugados, microchipados e recebem medicamentos para controle de pulgas e carrapatos, de acordo com a legislação vigente." %}

{% trans "Os animais são vacinados?" %}

{% trans "Sim. São vacinados de acordo com sua idade e condições." %}

{% trans "Quais são os documentos necessários para uma adoção?" %}

{% trans "RG, CPF e comprovante de endereço recente. É necessário apresentar RG original também no dia da retirada do pet." %}

{% trans "OBS: a equipe Salve Pets somente se responsabiliza por estabelecer a conexão entre o adotante e a instituição de adoção de pets, não exigindo portanto, nenhum documento no processo de manifestar interesse de adoção de um pet." %}

{% trans "Fui aprovado, posso retirar o pet no mesmo dia?" %}

{% trans "Sim. Após a aprovação da instituição para a adoção do pet, o adotante tem até 24h para a retirada do animal na instituição." %}

{% trans "Os animais para adoção são animais sociáveis?" %}

{% trans "Todos os animais passam por uma avaliação, mas cada pet tem a sua particularidade/personalidade. Esses animais são oriundos da rua e podem estar assustados, porém, enquanto estão no espaço da instituição de adoção, os pets realizam diversas atividades físicas, passam por técnicas de comportamento, estimulação cognitiva e sensorial (percepção, memória e raciocínio), entre outras atividades. Importante seguir as diretrizes repassadas pela instituição, bem como as orientações repassadas para apoiar o processo de adaptação do pet durante o período de pós-adoção." %}

{% trans "Posso sair com o meu cão no colo?" %}

{% trans "Sim, caso o cão já tenha tomado todas as vacinas. Para maior segurança, é indicado que seja adquirida uma coleira e uma guia para a saída do pet para a rua. No caso de filhotes que ainda não possuem todas as vacinas, orientamos o uso de caixa de transporte." %}

{% trans "Posso sair com o meu gato no colo?" %}

{% trans "Pensando no bem-estar e segurança do seu gato, é necessário levar uma caixa de transporte no dia da retirada." %}

{% trans "Tem alguma taxa para a adoção?" %}

{% trans "Algumas instituições solicitam uma colaboração para custear o gasto que tiveram com o animal adotado, a fim de conseguir realizar novos resgates. Esta cobrança fica a critério de cada instituição e o valor varia conforme definição de cada instituição parceira. O valor é estipulado por cada instituição e só será cobrado mediante aprovação do adotante na entrevista e posterior assinatura e aceite no contrato de adoção. Reforçamos que esta cobrança é paga diretamente à instituição parceira no ato da adoção, sem qualquer envolvimento da equipe Salve Pets, ou seja, a Salve Pets não interfere ou recebe parte alguma destes valores." %}

{% trans "Como posso encontrar meu pet perdido?" %}

{% trans "Para te ajudar na busca do seu pet perdido, realizamos, por meio de um processo automatizado, correspondências entre pets cadastrados como perdidos ou encontrados, considerando a localização de perda ou encontro, espécie, raça, porte e entre outras variáveis. Para isso, você deve cadastrar seu pet perdido no serviço" %} {% trans "Cadastrar Pet Perdido ou Encontrado" %}, {% trans "fornecendo o máximo de detalhes possível. Quando um pet for cadastrado como encontrado e a localização, espécie e porte forem similares ao do seu pet perdido, você receberá um e-mail avisando sobre a possibilidade de encontro do seu pet." %}

{% trans "Como posso cadastrar um pet encontrado?" %}

{% trans "Ao encontrar um pet perdido, para facilitar o encontro de seu tutor, cadastre-o no serviço" %} {% trans "Cadastrar Pet Perdido ou Encontrado" %} {% trans "indicando no campo Situação do pet o status de Encontrado.
